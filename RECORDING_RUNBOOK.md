# Recording Runbook

## Goal

Record a clear 2-5 minute video for the Kaching Football WC Campaign bounty.

The video must include:

- explanation of Kaching
- Football WC campaign breakdown
- live walkthrough from Kaching.vip
- wallet connect / sign up
- add funds
- buy at least one ticket
- referral link on screen and in description

## Before Recording

Prepare:

- Kaching referral link from `My Referral`
- Solana wallet with enough funds for at least one ticket
- small SOL balance for transaction fees
- USDC or supported token if required by Kaching
- clean browser window
- no private seed phrase / recovery phrase visible anywhere

## Recording Steps

1. Start recording.
2. Show title / intro screen with referral link.
3. Open `https://kaching.vip/`.
4. Point out:
   - Football WC / FIFA WC 2026 draw
   - pot size
   - ticket price
   - countdown
   - slots left
   - leaderboard / ticket dashboard areas
5. Click login / connect wallet.
6. User personally approves wallet connection popup.
7. Show wallet / funds area.
8. Explain what funds are needed:
   - ticket currency shown by Kaching
   - SOL for network fees
9. Add funds if required.
10. Return to Football draw.
11. Click Buy Ticket.
12. Select at least one ticket.
13. Review price.
14. User personally approves transaction popup.
15. Show My Tickets Dashboard / ticket confirmation.
16. Show where View Result / View OnChain can be checked later.
17. End with referral link CTA.

## If Purchase Cannot Be Completed

Do not fake a purchase.

Say clearly in the recording:

```text
For security, I am stopping before final transaction confirmation here. In the final step, you review the ticket cost in your wallet and approve the transaction yourself.
```

This is weaker than the official requirement, because the bounty asks for buying at least one ticket.

## Recommended Recording Format

- Horizontal 16:9
- 1080p if possible
- clear microphone
- browser zoom 100-125%
- no music louder than voice
- keep wallet popups readable, but never reveal seed phrases
